
# Running the Application

`` Using mvnw ``

### Make mvnw Executable

`` chmod +x mvnw ``

## Run the Application
`` ./mvnw spring-boot:run ``

## If you want to use specific profile
`` ./mvnw spring-boot:run -Dspring-boot.run.profiles=yourProfile ``

## Using Env Variables
`` export SPRING_PROFILES_ACTIVE=yourProfile ``
`` ./mvnw spring-boot:run ``
